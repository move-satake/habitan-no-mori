# 🌱 Habitan no Mori - Compassionate Habit Tracker

> *Your compassionate habit companion. It's OK to give up. There's no room for being blamed here. We keep supporting you until you become a friend with your habits. Life's getting better.*

## ✨ Features

- **One-Click Tracking**: Simply click the circle next to any habit to mark it as complete - that's it!
- **Compassionate Approach**: No guilt, no shame, just gentle encouragement
- **Super User-Friendly**: Clean, intuitive interface that anyone can use
- **Daily Focus**: Check your daily behaviors with ease
- **Encouraging Messages**: Positive reinforcement to keep you motivated
- **Local Storage**: Your data stays private on your device
- **Responsive Design**: Works perfectly on desktop and mobile

## 🚀 How to Use

1. **Add a Habit**: Type a habit name (like "Drink water" or "Take a walk") and click "Add Habit"
2. **Track Daily**: Click the circle next to any habit to mark it complete for today
3. **Stay Motivated**: Read the encouraging messages and see your progress stats
4. **No Pressure**: Completed habits turn green with a checkmark, but there's no judgment if you miss a day

## 🛠️ Technical Details

This is a simple, self-contained web application built with:
- Pure HTML5, CSS3, and JavaScript (no frameworks needed)
- Local Storage for data persistence
- Responsive design for all devices
- Progressive Web App ready

## 🌐 Getting Started

### Option 1: Direct Use
1. Download or clone this repository
2. Open `index.html` in any modern web browser
3. Start tracking your habits immediately!

### Option 2: Local Server
```bash
# Navigate to the project directory
cd habitan-no-mori

# Start a simple HTTP server
python3 -m http.server 8000
# or
npx serve .

# Open http://localhost:8000 in your browser
```

## 💝 Philosophy

Habitan no Mori believes in gentle, compassionate habit building. We understand that:
- Progress isn't always linear
- It's okay to miss days
- Small steps matter more than perfection
- Everyone deserves encouragement, not judgment
